# PiCK CLI

PiCK 학생 사용자를 위한 간단한 CLI 도구입니다.

## 설치

```bash
npm install -g pick-cli
```

## 사용법

```bash
pick
```

### 업데이트

```bash
# 수동 업데이트 확인 및 실행
pick --update
# 또는
pick -u
```

- 앱 시작 시 새 버전을 확인하고, 자동 업데이트가 켜져 있으면 백그라운드에서 업데이트를 시작합니다.
- 자동 업데이트 여부는 설정 메뉴에서 변경할 수 있습니다.

## 기능

- 로그인 및 자동 로그인
- 자동 업데이트
- 날짜별 급식 조회
- 주말 급식 신청
- 시간표 조회
- 조기 귀가 신청
- 외출 신청
- 교실 이동 신청
- 설정 관리

## 요구사항

- Node.js >= 18.0.0
- `npm install -g pick-cli` 방식으로 설치된 환경
