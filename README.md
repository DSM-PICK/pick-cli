# PiCK CLI

PiCK 학생 사용자를 위한 빠르고 간단한 CLI 앱

## 설치

```bash
npm install -g pick-cli
```

## 사용법

```bash
pick
```

### 업데이트

```bash
# 수동 업데이트 확인 및 실행
pick --update
# 또는
pick -u
```

## 기능

- 🔐 로그인/인증
- 📅 시간표 조회
- 🍽️ 주말 급식 신청
- 🏃 조기 귀가 신청
- 📝 외출 신청
- 🏫 교실 이동 신청
- ⚙️ 설정 관리

## 요구사항

- Node.js >= 21.0.0
